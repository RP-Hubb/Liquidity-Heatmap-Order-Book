This project is a production-grade Streamlit dashboard that simulates and visualizes a realistic 3D limit order book for financial markets. It features:
- Realistic order book simulation (normal, imbalance, spoofing, flash crash)
- Animated 3D Plotly surface with neon bid/ask coloring
- Dark institutional UI
- Interactive controls for market parameters
- Real-time liquidity and imbalance metrics

### How to Fork

1. Click the "Fork" button on the top right of this repository in GitHub.
2. Clone your forked repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_FORKED_REPO.git
   ```
3. Install dependencies:
   ```bash
   pip install streamlit numpy plotly
   ```
4. Run the app:
   ```bash
   streamlit run streamlit_3d_orderbook_lab.py
   ```
